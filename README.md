#  House Price Predictor

A simple Machine Learning project that predicts house prices based on:
- City
- Area (sq.ft)
- Number of Bedrooms

## Features
- Data preprocessing using **Pandas + Scikit-learn**
- Encoded categorical data (city names)
- Random Forest Regressor for prediction
- Handles unseen city names gracefully

## Files
| File | Description |
|------|--------------|
| `House price pthon.py` | Full training + model saving script |
| `House Price Predictor.ipynb` | Jupyter Notebook version |
| `House_model.joblib` | Trained model |
| `house_prices_data.csv` | Dataset |
